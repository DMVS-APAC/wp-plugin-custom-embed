== Description ==

Hi there! This is the official Dailymotion beta WordPress plugin! We are glad that you take part in this project and hope you will help us improve the plugin.

The plugin will support your editorial workflow by facilitating the embed of Dailymotion videos. You will also be able to import your playback settings from your Dailymotion account.
